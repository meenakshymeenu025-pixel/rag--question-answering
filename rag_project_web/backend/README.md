# RAG Project Web

## Setup

1. Create and activate a virtual environment.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the app:
   ```bash
   python app.py
   ```

## Environment variables

- UPLOAD_FOLDER
- VECTOR_STORE
- EMBEDDING_MODEL
- OLLAMA_MODEL
- CHUNK_SIZE
- CHUNK_OVERLAP
- MAX_CONTEXT_LENGTH
- TOP_K
